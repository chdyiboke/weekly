## 发布

发布一个npm包，首先需要一个账号

1. 然后登录
```
npm login

Username: ma_memg       //输入用户名
Password:               //输入密码
Email: (this IS public) //输入注册邮箱

```
不放心可以用一下命令查一下当前npm账户是谁

```
npm who am i
```

2. 发布流程是先npm run build 然后 npm publish

3. 上传完成之后， www.npmjs.com 查看有没有发布成功
